# craft it!
Social deduction / trivia crafting minigame for Minecraft 1.16+

# gamerules
- [ ] extra elements provided (stonecutter, smithing table)
    - `extra_workstations`
- [ ] free furnace fuel
    - `free_fuel`
- [ ] randomised items
    - `random_items`
    - 0: just random items, 1: hybrid, 2: only player-provided items, 3: manual
